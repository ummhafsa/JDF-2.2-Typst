# JDF 2.2 for Typst

An adaptation of the [Joyner Document Format (JDF) v2.2](https://www.overleaf.com/latex/templates/joyner-document-format-v2-dot-2/xysfjgnvxnqq) Overleaf template to [Typst](https://typst.app/), a modern document typesetting ecosystem.

> [!WARNING]
> This template is **unofficial** as of the time of writing. Students intending to use it for coursework **must** obtain approval from their course staff beforehand.

## Background

I made the switch to Typst before Overleaf cut down its compile times — and I'm glad I did, as Overleaf is presently next to unusable for any but the simplest documents. Anticipating that others may make a similar transition, I adapted the JDF template accordingly.

## What's Included

- **Template** — the JDF 2.2 layout, reimplemented in Typst
- **Sample document** — a ready-made example to get you started
- **Citation format** — fully worked out (shoutout to @a8712a for resolving that particular quandary!)

## Status

The output is not pixel-perfect compared to the LaTeX original, but it is close enough to be satisfactory. Contributions and refinements are welcome.

## Usage

Clone the repository and open the project in Typst:

```bash
git clone https://github.com/ummhafsa/JDF-2.2-Typst.git
```
