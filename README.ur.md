# JDF 2.2 برائے Typst

یہ Overleaf پر موجود [Joyner Document Format (JDF) v2.2](https://www.overleaf.com/latex/templates/joyner-document-format-v2-dot-2/xysfjgnvxnqq) کے سانچے کی [Typst](https://typst.app/) کے لیے تشکیلِ نو ہے — جو دستاویز کی تدوین کا ایک جدید نظام ہے۔

> [!WARNING]
> یہ سانچہ فی الحال **غیر رسمی** ہے۔ جو طلبہ اسے اپنے نصابی کام میں استعمال کرنا چاہیں، انھیں پہلے اپنے نصاب کے عملے سے اجازت لینا لازم ہے۔

## پسِ منظر

میں نے Overleaf کے ترجمائی اوقات میں کمی سے پہلے ہی Typst کی طرف منتقلی کر لی تھی — اور اچھا ہی کیا، کیونکہ اب Overleaf سادہ ترین دستاویزات کے سوا کسی کام کی نہیں رہی۔ یہ سوچ کر کہ دوسرے بھی اسی راہ پر آ سکتے ہیں، میں نے یہ سانچہ Typst کے لیے ڈھال لیا۔

## ذخیرے میں کیا ہے؟

- **سانچہ** — JDF 2.2 کی ترتیب، Typst میں از سرِ نو تعمیر شدہ
- **نمونہ دستاویز** — شروع کرنے کے لیے ایک تیار مثال
- **حوالہ جات کی ترتیب** — مکمل طور پر حل شدہ (اس الجھن کو سلجھانے پر @a8712a کا شکریہ!)

## موجودہ کیفیت

نتائج اصل LaTeX سانچے سے نقطہ بہ نقطہ تو مطابق نہیں، مگر اطمینان بخش حد تک قریب ہیں۔ بہتری کی تجاویز اور شراکت کا خیر مقدم ہے۔

## استعمال

ذخیرے کی نقل بنائیں اور دستاویز کو Typst میں کھولیں:
